# test

HTML/CSS로 만든 간단한 개인 포트폴리오 페이지입니다.

## 프로젝트 목적

자기소개, 기술 스택, 프로젝트 목록, 연락 링크를 한 화면에 보여주는 정적 포트폴리오 예제입니다. 현재 콘텐츠는 샘플 텍스트와 placeholder 이메일을 포함합니다.

## 기술 스택

- HTML
- CSS
- JavaScript placeholder

## 아키텍처

`index.html` 하나에 마크업과 스타일이 함께 들어 있는 단일 페이지 구조입니다. 별도 빌드 도구, 백엔드, 데이터 저장소는 없습니다.

## 폴더 구조

```text
.
├── index.html
└── README.md
```

## 실행 방법

브라우저에서 `index.html`을 직접 열거나 로컬 서버를 실행합니다.

```bash
python -m http.server 8000
```

## 환경변수

필요한 환경변수는 없습니다.

## 보안 및 공개 메모

secret/API key/password/token은 발견되지 않았습니다. 공개 저장소로 유지 가능합니다. 실제 포트폴리오로 쓰려면 placeholder 이메일과 샘플 프로젝트를 실제 정보로 교체하세요.
